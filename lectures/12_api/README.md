## Lecture plan

Learn RESTful API server structure and how to use it.
And also learn how to use popular libraries on server-side for example "momentjs" and "lodash".

### Learning objectives

1. RESTful API server design
2. Authentication
3. Cryptography
4. Session
5. HTTP cookie
6. HTTP request
7. JSON Web Tokens
8. Validation
7. Moment.js
7. Lodash.js
